Preguntas Frecuentes


¿Qué es WinSOL?
Es una aplicación web que nos permite generar CDs y DVDs de software libre para Windows de manera rápida y fácil, únicamente utilizando un PC y una conexión a Internet.
¿Qué es un fichero ISO?
Es un fichero con un formato compatible con aplicaciones de grabación tipo: K3B, Nero, EasyCD, etc. Aunque es un sólo archivo, una vez grabado, el CD/DVD contendrá la estructura de carpetas y archivos incluidos en la imagen ISO.
¿Cómo genero una composición?
El proceso está descrito brevemente en la Introducción y puedes obtener información más detallada en el manual incluido en la opción Ayuda.
¿Cómo puedo añadir más aplicaciones?
El proceso está descrito brevemente en la opción Añadir Aplicación y puedes obtener información más detallada en el manual incluido en la opción Ayuda.
¿Cómo grabo la imagen?
Abre la aplicación de grabación que utilices en tu equipo. Busca la opción "Grabar Imagen" o similar, seleccionando si es necesario, archivos con extensión ".iso". Selecciona la imagen que has descargado de WINSOL y grábala con las opciones predeterminadas.
¿Cómo utilizo el CD/DVD?
Los CD/DVDs generados a partir de las imágenes de WinSOL tienen un formato autoarrancable. De este modo, para utilizar el CD/DVD simplemente tendrás que introducirlo en un equipo con Windows y comprobarás como, automáticamente, se abre un navegador mostrando la página principal de tu compilación.
¿Cuánto tiempo tengo para descargar una imagen?
Aunque el tiempo máximo es determinado por el número de imágenes y la capacidad del servidor WinSOL. La aplicación tiene establecido, de momento, un mínimo de dos días para que tengas tiempo de descargar la imagen a tu equipo.
